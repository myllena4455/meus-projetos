Bem-vindo ao meu portfólio! Este é um espaço onde compartilho meus projetos, habilidades e experiências como desenvolvedora.

## 🎯 Sobre

Sou uma desenvolvedora apaixonada por criar soluções inovadoras e interfaces intuitivas. Este portfólio apresenta alguns dos meus projetos mais importantes, demonstrando minhas habilidades em desenvolvimento web, design responsivo e user experience.

## 🚀 Projetos em Destaque

### 1. **Site Setembro Amarelo**
- **Descrição**: Site informativo e interativo voltado à conscientização sobre saúde mental e prevenção ao suicídio
- **Tecnologias**: HTML, CSS
- **Objetivo**: Promover a conscientização sobre a importância do cuidado emocional

### 2. **Caixa Registradora Personalizada**
Sistema de Ponto de Venda (PDV) - LK Imports. Solução web projetada para agilizar vendas e gestão de estoque.
- **Tecnologias**: JavaScript, HTML, CSS
- **Foco**: Automatização de processos comerciais

### 3. **Conny - Rede Social para Esportes**
O Conecta+ (Saúde em Movimento) é uma plataforma digital inovadora focada na promoção de bem-estar e integração social através da atividade física.
- **Tecnologias**: React, Node.js, MongoDB
- **Objetivo**: Conectar pessoas interessadas em atividades físicas e bem-estar

### 4. **O Anel dos 4 Reinos (RPG)**
Um RPG de aventura e estratégia desenvolvido inteiramente em JavaScript Vanilla.
- **Tecnologias**: JavaScript Vanilla, HTML, CSS
- **Destaque**: Lógica de jogo complexa, animações suaves
