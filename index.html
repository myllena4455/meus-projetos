Sistema de Caixa

Ferramenta: Frontend: HTML, CSS, JavaScript (vanilla) |Backend: Node.js + Express|Banco: SQLite (arquivo local database.sqlite)

Objetivo: O objetivo central deste sistema é oferecer uma ferramenta robusta e intuitiva que transforma um computador em um terminal de vendas eficiente, permitindo o controle total sobre transações, inventário e desempenho.

Descrição: O sistema oferece cadastro e busca rápida de produtos, controle de estoque com baixa automática no fechamento da venda, campos de desconto/valor adicional/observações no checkout, exportação de relatórios, impressão de recibos em PDF e registro de vendas em banco SQLite local. Ideal para comércios pequenos que precisam de um PDV local, sem dependência de nuvem.

Destaques:

  • Gestão de Estoque e Catálogo: Funções para Importar/Exportar dados, Cadastrar novos produtos e visualizar o Estoque e Preço em tempo real.
  • Operação de Vendas (Carrinho): Adiciona produtos por código ou nome, totaliza a compra e permite Fechar Venda e Imprimir Recibo (nota não fiscal).
  • Controle de Caixa: Exibe as Vendas do Dia e possui ferramentas para Gerar Relatório e Finalizar Dia.
  • Funcionalidade Adicional: Botão Config Loja para ajustes do sistema.

Rpg O Anel dos 4 Reinos

Ferramenta: JavaScript, css e html

Objetivo: objetivo central do projeto O Anel dos Quatro Reinos é derrotar os chefes de cada uma das oito fases para coletar seus anéis únicos e enfrentar o Abominável Guardião Final para salvar os reinos. ​Para alcançar esse objetivo final, o jogador deve seguir uma estrutura de progresso e sobrevivência: ​Progressão de Fases: É necessário derrotar três inimigos comuns para atrair o Boss da fase. ​Coleta de Itens: Ao vencer cada Boss, o jogador ganha um anel único que deve ser equipado para acumular buffs permanentes de atributos como ataque e defesa. ​Gestão de Recursos: O jogador deve utilizar o sistema de compras para adquirir consumíveis e equipamentos que facilitam o avanço até as fases finais. ​Desafio de Lore: É obrigatório completar um quiz sobre a história dos reinos para ganhar o direito de enfrentar o último desafio. ​Sobrevivência: Gerenciar o sistema de 3 vidas, evitando o Game Over final que ocorre na quarta morte.

Descrição: ​O Anel dos Quatro Reinos é um RPG de aventura e estratégia desenvolvido inteiramente em JavaScript Vanilla. O projeto foca em mecânicas clássicas de progressão e combate, desafiando o jogador a derrotar chefes poderosos para restaurar o equilíbrio dos reinos.

Destaques:

  • Engine Própria: Lógica de combate, sistema de turnos e gerenciamento de status desenvolvidos sem o uso de frameworks externos.
  • Arquitetura de Classes: 11 classes distintas (Arqueira, Necromante, Tank, etc.) com habilidades e atributos únicos balanceados via código.
  • Persistência de Dados: Sistema de save/load utilizando LocalStorage com suporte para até 5 slots de progresso.
  • Narrativa Integrada: Lore expandida através de diálogos e um Quiz de conhecimento que desbloqueia o desafio final
